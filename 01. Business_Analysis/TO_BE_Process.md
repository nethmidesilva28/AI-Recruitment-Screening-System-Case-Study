# TO-BE Process (AI-Powered Recruitment Workflow)

The TO-BE process introduces an AI-powered recruitment screening system that automates early-stage candidate evaluation.

---

## Process Flow

1. Job requisition is created
2. Job description is processed by AI system
3. Candidates submit CVs
4. AI system parses CVs into structured data
5. AI matches candidates against job requirements
6. Candidates are scored and ranked
7. Recruiters review AI-ranked shortlist
8. Recruiters approve or override selections
9. Hiring Managers finalize interview list

---

## Key Improvements

- Automated CV screening using AI
- Standardized candidate scoring system
- Faster shortlisting process
- Reduced human bias in early screening
- Data-driven recruitment decisions
