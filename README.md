# AI Recruitment Screening System – Case Study

## Overview
This project presents a case study for an AI-powered Recruitment Screening System designed to optimize hiring efficiency, reduce bias, and improve candidate-job matching accuracy.

## Problem Statement
Recruitment teams spend excessive time manually screening large volumes of CVs, resulting in delays, inconsistent evaluation, and potential loss of qualified candidates.

## Solution
An AI-driven screening system that:
- Parses CVs into structured data
- Matches candidates with job descriptions
- Generates ranked candidate lists
- Supports recruiter decision-making with dashboards

## Key Focus Areas
- Business Analysis (BRD, SRS, Stakeholders, Process Modeling)
- Product Management (Roadmap, MVP, Backlog, Sprints)
- AI System Design (Matching logic, scoring, bias mitigation)
