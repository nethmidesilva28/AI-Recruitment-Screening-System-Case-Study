# Product Roadmap

---

## Phase 1: MVP Development (Month 1–2)
- CV parsing engine
- Job description parser
- Basic AI matching logic
- Simple recruiter dashboard

---

## Phase 2: Enhancement (Month 3–4)
- Improved AI scoring model (semantic matching)
- Advanced filtering options
- Better UI/UX for dashboard
- Performance optimization

---

## Phase 3: Intelligence Layer (Month 5–6)
- Bias detection system
- Candidate recommendation engine
- Analytics dashboard for hiring trends

---

## Phase 4: Enterprise Expansion (Future)
- ATS integrations
- Interview scheduling automation
- Multi-role recruitment support
- API ecosystem for external systems
