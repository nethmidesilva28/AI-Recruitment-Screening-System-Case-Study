# Epics, Features & Product Backlog

---

## EPIC 1: CV Processing System

### Features:
- Upload CV
- Parse CV content
- Extract structured data

### User Story:
- As a recruiter, I want CVs to be automatically structured so I don’t need to manually read each one.

---

## EPIC 2: Job Processing System

### Features:
- Input job description
- Extract required skills
- Normalize job requirements

---

## EPIC 3: AI Matching Engine

### Features:
- Match CV with job description
- Generate relevance score
- Apply weighting logic

---

## EPIC 4: Candidate Ranking System

### Features:
- Rank candidates
- Filter by score threshold
- Sort results dynamically

---

## EPIC 5: Recruiter Dashboard

### Features:
- View candidates
- Filter & search candidates
- Override AI decision
- Export shortlist
