# Acceptance Criteria

---

## CV Upload
- Given a valid CV file, when uploaded, then system extracts structured data successfully

---

## Job Parsing
- Given a job description, when processed, then system extracts required skills and experience

---

## Candidate Matching
- Given a candidate profile and job description, when processed, then system generates a relevance score

---

## Ranking System
- Given multiple candidates, when processed, then system ranks them in descending order of score

---

## Recruiter Dashboard
- Given ranked candidates, when displayed, then recruiter can filter, sort, and override results
