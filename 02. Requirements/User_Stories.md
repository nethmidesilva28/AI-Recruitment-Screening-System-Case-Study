# User Stories

---

## Recruiter

- As a recruiter, I want to upload job descriptions so that the system can analyze requirements.
- As a recruiter, I want to view ranked candidates so that I can shortlist efficiently.
- As a recruiter, I want to override AI decisions so that I retain control over hiring.

---

## Hiring Manager

- As a hiring manager, I want to define job requirements so that the system finds relevant candidates.
- As a hiring manager, I want to approve shortlisted candidates so that quality is maintained.

---

## Candidate

- As a candidate, I want my CV to be fairly evaluated so that I have equal opportunity.
- As a candidate, I want to receive timely feedback so that I stay informed.
