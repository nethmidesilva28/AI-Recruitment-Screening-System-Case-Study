# Scope Definition

## In Scope

The system will include the following functionalities:

### Candidate Processing
- CV upload and parsing (PDF, DOCX, LinkedIn profiles)
- Extraction of structured candidate data (skills, experience, education)

### AI Matching System
- Job description analysis
- Candidate-job matching using AI scoring
- Candidate ranking and filtering

### Recruiter Tools
- Dashboard for viewing ranked candidates
- Filtering and sorting options
- Manual override of AI decisions

### Reporting
- Candidate shortlisting summaries
- Hiring funnel analytics

---

## Out of Scope

The following are excluded from the current scope:

- Full end-to-end Applicant Tracking System (ATS)
- Interview scheduling automation
- Background verification systems
- Payroll or HR management systems
- Video interview analysis tools

---

## System Boundaries

The system operates as a **decision-support layer** on top of existing recruitment platforms, not as a replacement for full HR systems.
